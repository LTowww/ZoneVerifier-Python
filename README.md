## 🇺🇸 English

### 📌 About the Project

**ZoneVerifier** is a simple Python application that reads a product code and determines the region of Brazil it belongs to — or whether the product is imported.

This project is a Python adaptation of an original version developed in Portugol, showcasing the transition to a more widely used programming language and improved code structure.

---

### 🚀 Features

* 📥 Reads user input (product code)
* 🗺️ Identifies product region based on code
* 🌎 Detects imported products
* 🧹 Clears terminal screen for better user experience
* 🧩 Uses `match-case` (Python 3.10+)

---

### 🧠 Region Mapping

| Code  | Region    |
| ----- | --------- |
| 1     | South     |
| 2     | North     |
| 3     | East      |
| 4     | West      |
| 5-6   | Northeast |
| 7-9   | Southeast |
| 10    | Midwest   |
| 11    | Northwest |
| Other | Imported  |

## 🇧🇷 Português

### 📌 Sobre o Projeto

O **ZoneVerifier** é uma aplicação simples em Python que lê o código de um produto e determina de qual região do Brasil ele vem — ou se é importado.

Este projeto é uma adaptação em Python de uma versão original desenvolvida em Portugol, mostrando a evolução para uma linguagem mais utilizada no mercado e uma melhor organização do código.

---

### 🚀 Funcionalidades

* 📥 Leitura de entrada do usuário (código do produto)
* 🗺️ Identificação da região do produto
* 🌎 Detecção de produtos importados
* 🧹 Limpeza da tela para melhor experiência
* 🧩 Uso de `match-case` (Python 3.10+)

---
